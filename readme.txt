Really Simple and Developer Friendly Testimonials.

== Description ==

You can start collecting your endorsements or reviews in just a few simple steps. Both novices and experts will value the plethora of adaptable features developed over four years in response to user feedback and requests.

== Installation ==

= Installation from within WordPress =

1. Visit **Plugins > Add New**.
2. Search for **Really Simple Testimonials**.
3. Install and activate the Performance Lab plugin.

= Manual installation =

1. Upload the entire `performance-lab` folder to the `/wp-content/plugins/` directory.
2. Visit **Plugins**.
3. Activate the Testimonials.
